##  README (English)
##  Nature Conservation Landing Page
This is a responsive landing page dedicated to environmental protection and wildlife conservation. The project presents an organization’s mission, activities, and achievements, and encourages users to participate or donate.

### Features:
 + Fully responsive layout using HTML5, CSS3, and Bootstrap 4
 + Clean and modern design with a nature-oriented aesthetic
 + Sections:
   1. Navigation bar with logo and CTA button
   2. Hero section with background image and call to action
   3. Mission and description with image
   4. Key statistics and achievements
   5. Work methodology
   6. Donation section with call to support
   7. Footer with useful links and contact information

##  README (Русский)
## Лендинг по защите природы
Это адаптивный лендинг, посвящённый охране окружающей среды и защите дикой природы. Проект рассказывает о миссии, достижениях и деятельности организации, а также призывает к участию и пожертвованиям.

### Основные особенности:
 + Полностью адаптивная верстка с использованием HTML5, CSS3 и Bootstrap 4
 + Чистый и современный дизайн в природной тематике
 + Разделы:
   1. Навигация с логотипом и кнопкой
   2. Главный экран с фоновым изображением и призывом к действию
   3. Миссия с изображением и описанием
   4. Основные достижения и цифры
   5. Как мы работаем — список шагов
   6. Блок пожертвования
   7. Футер с ссылками и контактами
